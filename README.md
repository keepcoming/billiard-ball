# billiard-ball
#### 开发环境
- IntelliJ IDEA
- MySQL 8.0
- Java 8+
- 微信小程序
- 开门、开灯等设备对接
- 微信支付
- 美团、抖音对接

#### 部署说明
涉及到PC端、微信小程序端、微信公众号端、设备管理端等如有需要请联系。


遇到问题可以直接在项目提issue，会尽可能回复。。。
